Packets that are being resent

Land 

Terrain data

Wind 

Wind data

Cloud 

Cloud data

Task 

This deals almost exclusively with the object updates and properties. It also does GenericMessages and KillObject packets as well.

This category may become saturated with packets if there are many objects in the sim, or if it is a very highly active sim (many moving objects).

Texture 

Texture assets

Asset 

Non-texture assets

State 

Avatar and primitive data

This category WILL be saturated with packets after a link or selecting a large object. So when assigning a packet to this category, be aware that after a link, packets will not be sent for some time in this category. This is a sub-category of Task

Transfer 

This handles info that the client uses to be able to function in the world. This includes packets like Chat and directory, group, and profile packets

This category shouldn't ever be extremely saturated with packets. This is a sub-category of Task

Count 

The number of packet categories to throttle on. If a throttle category is added or removed, this number must also change
