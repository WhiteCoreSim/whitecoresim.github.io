Property Documentation

int WhiteCore.ClientStack.TokenBucket.Content
get

The number of bytes that can be sent at this moment. This is the current number of tokens in the bucket

If this bucket has a parent bucket that does not have enough tokens for a request, <cref>RemoveTokens</cref>

will return false regardless of the content of this bucket

float WhiteCore.ClientStack.TokenBucket.DripPerMS
get

The speed limit of this bucket in bytes per millisecond

float WhiteCore.ClientStack.TokenBucket.DripRate
getset

The speed limit of this bucket in bytes per second. This is the number of tokens that are added to the bucket per second

Tokens are added to the bucket any time <cref>RemoveTokens</cref>

is called, at the granularity of the system tick interval (typically around 15-22ms)

int WhiteCore.ClientStack.TokenBucket.MaxBurst
getset

Maximum burst rate in bytes per second. This is the maximum number of tokens that can accumulate in the bucket at any one time

TokenBucket WhiteCore.ClientStack.TokenBucket.Parent
get

The parent bucket of this bucket, or null if this bucket has no parent. The parent bucket will limit the aggregate bandwidth of all of its children buckets
