Detailed Description

Entry for a port of Bullet (http://bulletphysics.org/) to WhiteCore. This module interfaces to an unmanaged C++ library which makes the actual calls into the Bullet physics engine. The unmanaged library is found in opensim-libs::trunk/unmanaged/BulletSim/. The unmanaged library is compiled and linked statically with Bullet to create BulletSim.dll and libBulletSim.so (for both 32 and 64 bit).
